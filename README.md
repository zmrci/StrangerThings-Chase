In this game you can move around using the keyboard arrows, you are the white player and the demogorgon is the black player. 
The demogorgon can spawn randomly from one of the spawn points, which are painted yellow. The spawn points are also spawned randomly in the four corners.
the demogorgon chases in you in a given intervallum and that stops and disapears, rests and then chases again and so on.
